# Toppsample
## 参数说明
- **OpName**：ToppsampleOperation
- **PARAM**
**Param Type**：OpParam::Toppsample
 
| 名称  | 类型  | 描述 |
| ------------ | ------------ | ------------ |
|randSeed|uint32_t||
 
- **In/Out Tensor**
 
|名称 | 类型  | dims  | dtype  |
| ------------ | ------------ | ------------ | ------------ |
|Cumsumed_prob|In	|[batch, vocal_size]|float16|
|topp|In|[batch,1]|	float16	|
|idx|Out	|[batch,1]|	float16|	
 
## 功能描述
- 算子功能：后处理算子topk,topp融合
